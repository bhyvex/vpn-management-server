Download setup wizard

You can either manually create CSR and use this web wizard or download native wizard for Mac OS X, Linux or Windows. In addition to this software, you must install OpenVPN.

  • Mac OS X
  • Windows (when opening, right click and select "Run as administrator")
  • Linux

Manual way (not recommended)

You can create new VPN certificate using these web pages. Using wizard (see above) is much easier way. Please read VPN instructions from Confluence page.

There is following steps: creating valid CSR (Certificate Signing Request); uploading CSR; confirming CSR data; confirming identity and finally downloading certificate file.

In your computer you have to install OpenVPN, generate private key and CSR and add Futurice configurations (provided in zip file in the end of this wizard).
