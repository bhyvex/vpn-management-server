Download setup wizard

You can either manually create CSR and use this web wizard or download native wizard for Mac OS X, Linux or Windows. In addition to this software, you must install OpenVPN.

  • Mac OS X
  • Windows (when opening, right click and select "Run as administrator")
  • Linux

Manual way

Only use this web wizard if you can't use native wizard from above.

If you can't use easier setup wizard, you can create a new VPN certificate with these web pages. Using wizard (see above) is much easier. Please read VPN instructions from Confluence page.

This web wizard have following steps:

  1. Filling in your information
  2. Creating and uploading CSR (Certificate Signing Request)
  3. Confirming CSR data
  4. Validating your identity (you'll need your mobile phone for this)
  5. Downloading certificates and configuration files

You have to install OpenVPN, generate a private key and CSR. Finally, after finishing this web wizard, you have to import configuration files to your OpenVPN before you can actually use the configuration.
